# My-Bucket-List
Telegram bot for creating and managing lists of Wishes.

A cache layer has been created on top of the database, which reduces the number of queries to the database.

## Used Technologies
* Java 1.8
* Maven
* Telegram Bots API
* JDBC for MySQL
* JUnit
